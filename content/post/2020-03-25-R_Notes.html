Matrices in R!

We can start by creating a 3X3 matrix and filling it with the numbers 1 to 9:

A <- matrix(1:9, nrow = 3, ncol = 3)

If we use the class() function to check the class it should be of the matrix class:

class(A)
## [1] "matrix"
